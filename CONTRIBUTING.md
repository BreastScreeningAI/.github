# Contributing

Thank you for your interest in BreastScreening-AI. This organization hosts
project materials related to AI-assisted breast screening research, prototypes,
and public communication.

## Before You Start

- Open an issue or discussion before proposing substantial changes.
- Keep patient privacy, clinical safety, and reproducibility in mind.
- Do not upload patient-identifiable data, credentials, private datasets, or
  confidential clinical material.
- Check each repository for its own contribution instructions. Repository-local
  guidance takes precedence over this default file.

## Pull Request Guidelines

- Keep changes focused and easy to review.
- Explain the motivation, scope, and validation performed.
- Include screenshots for public website or interface changes when relevant.
- Add or update documentation when behavior, assumptions, or setup steps change.
- Avoid unrelated formatting churn.

## Research and Clinical Context

Unless a repository explicitly states otherwise, code and materials in this
organization should be treated as research or project communication artifacts.
They are not a substitute for professional clinical judgement and should not be
used for patient care without appropriate validation, governance, and approval.

## Reporting Problems

For bugs or documentation issues, open a GitHub issue in the relevant
repository. For security or privacy concerns, follow `SECURITY.md` instead of
opening a public issue.
