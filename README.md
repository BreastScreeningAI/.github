# BreastScreening-AI GitHub Profile

This repository contains the default GitHub organization profile and community
health files for the BreastScreening-AI organization.

## Repository Layout

- `profile/README.md` controls the public organization profile shown at
  `github.com/BreastScreeningAI`.
- `CONTRIBUTING.md`, `SECURITY.md`, and `SUPPORT.md` provide default guidance
  for repositories in the organization unless a specific repository overrides
  them.
- `LICENSE` applies to the contents of this repository.

## Public Website

The public project website is maintained separately in the
`breastscreeningai.github.io` repository and is published at
https://breastscreeningai.eu.
