# Profile Maintenance Checklist

Use this checklist when updating the BreastScreening-AI organization profile or
default community health files.

## Profile README

- Confirm the header image still renders.
- Check that website links consistently use `https://breastscreeningai.github.io`.
- Check that repository links point to the correct GitHub repositories.
- Update focus areas if the project scope changes.
- Keep clinical safety language visible and conservative.
- Remove stale template text or inactive links.

## Organization Defaults

- Keep `CONTRIBUTING.md`, `SECURITY.md`, `SUPPORT.md`, and
  `CODE_OF_CONDUCT.md` aligned with repository practices.
- Add repository-specific guidance inside individual repositories when their
  needs differ from the organization defaults.
- Review issue and pull request templates after major workflow changes.

## Public Communication

- Confirm names, roles, partners, recognitions, and project pages against the
  public website before updating them here.
- Avoid publishing patient-identifiable data, credentials, private datasets, or
  confidential clinical material.
- Make clinical and AI claims carefully, with clear limits around research,
  prototype, and validated use.
